# StatMaster — Exam Prep

Fast **Probability & Statistics exam preparation**. Practice real exam-style problems (from your course PDF), revise short notes per topic, skip anything you already know.

No login. No database. Progress saves in your browser.

## Quick start

```bash
npm install
npm run dev
```

Open http://localhost:3000 → **Practice** or **Revise**

## What's inside

| Section | Purpose |
|---------|---------|
| **Practice** | 57 exam-style problems with hints + step-by-step solutions |
| **Revise** | Short notes per topic (~5 min each) — open any topic, any order |
| **Mock Exam** | 3 timed exams using the same question style |
| **Tools** | Formula sheet, Z-table, calculator |

## Deploy (Vercel)

No environment variables. Import repo → Deploy. See [DEPLOYMENT.md](./DEPLOYMENT.md).

## Scripts

- `npm run generate:important` — regenerate exam questions JSON
- `npm run validate:content` — validate content files
